🌐 Deployment
This project is deployed using Netlify.

To deploy your own version:

Fork this repo.

Push to your GitHub.

Go to Netlify → New Site from Git.

Connect your repository.

Set the build command to N/A (if static) and publish directory to /.

Deploy 🚀
